# Taller de Bingo - Programación Orientada a Objetos (POO)

Este proyecto es una simulación de un juego de Bingo automatizado, diseñado bajo el paradigma de orientación a objetos y estructurado en paquetes para una mejor organización y escalabilidad.

## Instrucciones de Ejecución

Para ejecutar el proyecto correctamente, siga estos pasos:

1. Asegúrese de estar en la carpeta raíz del proyecto (`bingo`) desde su terminal.
2. Ejecute el archivo principal utilizando el siguiente comando:
   ```bash
   python main.py
